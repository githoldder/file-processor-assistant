# 寒假居家哑铃增肌计划
**适用人群**：健身新手（瘦子增肌方向）
**训练频率**：每周 4 练，3 天休息
**所需器械**：一对哑铃、瑜伽垫、椅子/床沿
**核心理念**：复合动作为主，吃够睡好

## 一周训练日历
| 周次 | 周一 | 周二 | 周三 | 周四 | 周五 | 周六 | 周日 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| Day | Day 1 (胸+肩) | Day 2 (背+手臂+腹) | 休息 | Day 3 (腿) | Day 4 (肩+线条) | 休息 | 休息 |

## 每日训练内容
### 🔴 Day 1：胸部 + 肩部
**目标**：打造挺拔胸肌，拓宽肩膀，改善单薄身材

| 动作名称 | 组数建议 | 动作要点与呼吸 | 训练肌群 |
| ---- | ---- | ---- | ---- |
| 哑铃平板卧推 | 4组 × 10-12次 | 仰卧，大臂与身体呈 45 度，推起时呼气，下落时吸气 | 胸大肌 |
| 哑铃上斜卧推 | 3组 × 12次 | 背部垫高（靠在床或沙发上），倾斜 30-45 度 | 上胸 |
| 哑铃坐姿推举 | 3组 × 10-12次 | 坐姿或跪姿，腰背挺直，将哑铃从耳侧推至头顶上方 | 肩部 |
| 哑铃前/侧平举 | 3组 × 15次 | 前平举练肩膀前侧，侧平举练肩膀中束（宽肩关键），重量要轻 | 肩部前束、中束 |

### 🔵 Day 2：背部 + 手臂 + 腹部
**目标**：打造倒三角背部，雕刻腹肌线条，矫正体态

| 动作名称 | 组数建议 | 动作要点与呼吸 | 训练肌群 |
| ---- | ---- | ---- | ---- |
| 哑铃罗马尼亚硬拉 | 4组 × 12次 | 膝盖微屈，身体前倾，感受大腿后侧和下背部的拉伸 | 背阔肌、腘绳肌 |
| 哑铃单臂划船 | 3组 × 12次/边 | 一手扶固定物，背部挺直，手肘向后拉，夹紧背部 | 背阔肌 |
| 哑铃弯举 | 3组 × 12-15次 | 站姿，大臂夹紧身体，不要晃动借力 | 二头肌 |
| 哑铃俄罗斯转体 | 3组 × 20次 | 坐姿，手持哑铃在体前左右旋转，收紧核心 | 腹斜肌 |
| 平板支撑 | 3组 × 45秒 | 身体呈一条直线，不要塌腰，核心绷紧 | 核心肌群 |

### 🟢 Day 3：腿部训练
**目标**：刺激全身肌肉生长，提升代谢，促进食欲

| 动作名称 | 组数建议 | 动作要点与呼吸 | 训练肌群 |
| ---- | ---- | ---- | ---- |
| 哑铃高脚杯深蹲 | 4组 × 15次 | 双手捧一只哑铃于胸前，双脚比肩稍宽，蹲至大腿平行地面 | 股四头肌、臀部 |
| 哑铃箭步蹲 | 3组 × 12次/腿 | 前后脚站立，身体垂直下压，保持身体平衡 | 股四头肌、臀大肌 |
| 哑铃臀桥 | 3组 × 15次 | 仰卧，哑铃置于小腹，发力将髋部顶起，夹紧臀部 | 臀大肌 |
| 站姿提踵 | 3组 × 20次 | 手持哑铃，脚尖踩书或台阶，上下浮动脚后跟 | 小腿肌群 |

### 🟡 Day 4：肩部与线条
**目标**：强化肩后束，让肩膀饱满立体，手臂变粗

| 动作名称 | 组数建议 | 动作要点与呼吸 | 训练肌群 |
| ---- | ---- | ---- | ---- |
| 哑铃俯身飞鸟 | 4组 × 15次 | 身体前倾，背部平行地面，双手向两侧打开 | 肩后束 |
| 哑铃侧平举 | 4组 × 15次 | 保持肘部微屈，不要甩动 | 肩中束 |
| 哑铃颈后臂屈伸 | 3组 × 12次 | 单手或双手，哑铃举过头顶向后拉 | 三头肌 |
| 锤式弯举 | 3组 × 12次 | 掌心相对，像锤钉子一样 | 二头肌、前臂 |

## 💡 瘦子增肌“三板斧”
### 1. 饮食策略：多餐次，高热量
瘦子练不壮通常是因为吃不够
- 早餐：鸡蛋 + 牛奶 + 面包（必须吃饱）
- 加餐：训练前/后喝牛奶或吃香蕉
- 晚餐：多吃肉（鸡胸、牛肉、鱼肉），碳水（米饭、面条）不能少
- 夜宵：睡前一杯全脂牛奶或坚果
### 2. 训练技巧：慢控制，念动一致
家里重量有限，靠控制增加难度
- 离心收缩：举起用 1 秒，放下用 2-3 秒
- 顶峰收缩：动作最高点停顿 1 秒，感受肌肉收紧
### 3. 休息恢复：睡大觉
- 睡眠：每天必须睡够 7-8 小时
- 休息日：不要做高强度有氧，休息日用来长肉
