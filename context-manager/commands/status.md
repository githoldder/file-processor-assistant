# /context-manager:status

显示当前项目的完整状态摘要，包括进度、当前阶段、待办事项等。

## 用法

```
/context-manager:status
/context-manager:status [项目名]
```

## 参数

- `项目名` (可选): 指定项目名称。如不提供，自动检测当前目录下的项目。

## 输出内容

- 项目基本信息
- 完成进度可视化
- 当前阶段详情
- 待解决问题清单
- 最近活动记录
- 建议的下一步行动

## 触发条件

- 用户进入项目目录时自动提示
- 用户主动查询时
- 阶段切换后自动更新
