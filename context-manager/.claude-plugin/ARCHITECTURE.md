# Context Manager 架构设计文档

## 设计目标

1. **零上下文丢失**: 用户无需重复说明
2. **无缝接力**: Agent 之间完美交接
3. **熵减管理**: 自动维护项目整洁
4. **可恢复性**: 随时从任意点恢复

## 核心概念

### 接力跑协议 (Relay Protocol)

阶段间的标准交接流程：

```
Phase N                        Phase N+1
  │                              │
  ▼                              ▼
┌────────────┐            ┌────────────┐
│ 执行阶段任务 │            │ 读取 Handoff │
│ 生成产出    │──Handoff──→│ Document   │
│ 记录决策    │            │ 理解上下文  │
│ 跟踪问题    │            │ 继续工作    │
└────────────┘            └────────────┘
```

### 上下文三明治

每个阶段包裹在上下文中：

```
┌─────────────────────────────────────┐
│ 输入: Handoff Document (from prev)  │
├─────────────────────────────────────┤
│           阶段执行                   │
│  • 任务执行                          │
│  • 产出生成                          │
│  • 决策记录                          │
│  • 问题跟踪                          │
├─────────────────────────────────────┤
│ 输出: Handoff Document (to next)    │
└─────────────────────────────────────┘
```

┌─────────────────┐

│ Orchestrator    │
└────────┬────────┘
         │
    ┌────┴────┐
    ▼         ▼
┌────────┐ ┌────────┐

    ▼
